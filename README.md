# Border-waiting-times---Time-Series-Forecasting-with-LSTM

Mexico-USA border-waiting-time series forecasting using LSTM neural networks. The data set contains the waiting time and an UTC information format of different sentry lines across the border. 

The was collected from July to October 2017.
